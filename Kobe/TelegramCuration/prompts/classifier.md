   {
     "usable": true|false|"verify",

     "topic": "主题(可选)",

     "actions": ["动作1","动作2"],
     "confidence": 0.0-1.0
   }

4) 严格只输出 JSON，不要额外文字。

输入：messages（字符串数组，最多20条）。
输出：与输入等长的 JSON 结果数组。
